## German Tax Concepts Explained

| Term | English | Description |
|------|---------|-------------|
| Kaltmiete | Cold Rent | Base rent excluding utilities |
| Warmmiete | Warm Rent | Rent including allocable costs |
| Hausgeld | Service Charge | Monthly WEG payment |
| AfA | Depreciation | 2-3% annual building depreciation |
| Grunderwerbsteuer | Transfer Tax | 3.5-6.5% depending on state |
